# Basic Calculator

A simple calculator built with **HTML, CSS, and JavaScript**.  
This project was created to practice front‑end development fundamentals and demonstrate how to structure code across separate files for clarity and maintainability.

## Features
- Addition, subtraction, multiplication, division
- Backspace and clear (AC) functions
- Keyboard support (Enter key to calculate)
- Responsive layout with styled UI

## Project Structure
- `calculator.html` – main HTML file containing the calculator layout
- `cal_style.css` – stylesheet for design and layout
- `cal_js.js` – JavaScript logic for handling input and calculations
